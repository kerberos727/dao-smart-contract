# impact-market-token
Token and governance for Impact Market
## Install
```
nvm use 12
yarn --ignore-engines
``` 
## Build / Test / Deploy
`yarn build` / `yarn coverage` / `yarn deploy`
## Documentation
* Generate with `yarn docgen`
* Navigable HTML documentation from `./docs/index.html`
